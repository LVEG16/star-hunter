## 1. 产品概述
一款经典打飞机（射击）小游戏，玩家操控战机击落敌机、躲避弹幕、收集道具，追求高分挑战。支持玩家登录、全球排行榜和丰富个人资料统计。
- 面向休闲玩家，提供紧张刺激的街机射击体验
- 目标：打造视觉效果出色、操作流畅、可玩性强的网页射击游戏，具备社交竞争功能

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 游客 | 无需注册 | 浏览排行榜、查看他人资料 |
| 玩家 | 昵称登录 | 游戏游玩、提交分数、查看个人资料、参与排行榜 |

### 2.2 功能模块
1. **登录界面**：昵称输入、头像选择、本地身份生成
2. **游戏主界面**：开始游戏、操作说明、玩家信息、入口导航
3. **游戏界面**：战机操控、敌机生成、子弹发射、碰撞检测、道具系统、分数统计、详细数据追踪
4. **游戏结束界面**：最终得分、详细统计、分数上传、重新开始
5. **排行榜页面**：全球Top100、我的排名、前三名特效、查看他人资料
6. **个人资料页面**：详细统计、成就徽章、历史战绩

## 3. 核心流程
玩家首次进入登录界面，输入昵称选择头像后生成唯一ID保存到本地。进入主菜单后可开始游戏、查看排行榜或个人资料。游戏结束后自动上传分数到服务器，可跳转排行榜查看排名。

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深蓝/深紫太空背景，霓虹青色(#00FFD1)为主强调色，橙红(#FF4D4D)为危险色
- 排行榜排名色：金(#FFD700)、银(#C0C0C0)、铜(#CD7F32)
- 按钮风格：霓虹发光边框按钮，悬浮时光晕增强
- 字体：标题使用 Orbitron（科幻感），正文使用 Rajdhani
- 布局：居中Canvas游戏区域，HUD叠加显示
- 视觉风格：赛博朋克 + 太空射击，霓虹光效、粒子尾焰、星空滚动

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录界面 | 标题 | Orbitron标题，霓虹发光 |
| 登录界面 | 昵称输入框 | 霓虹边框输入框，focus时发光增强 |
| 登录界面 | 头像选择 | 6个预设头像网格，选中时霓虹边框 |
| 登录界面 | 进入按钮 | 霓虹发光边框按钮 |
| 游戏主界面 | 玩家信息卡 | 左上角，头像+昵称+最高分 |
| 游戏主界面 | 导航按钮 | 右上角，排行榜/个人资料图标按钮 |
| 排行榜页面 | 前三名展示 | 顶部特殊卡片，金银铜奖牌特效 |
| 排行榜页面 | 列表 | 玻璃拟态行，排名+头像+昵称+分数 |
| 排行榜页面 | 我的排名 | 底部固定，高亮霓虹边框 |
| 个人资料页面 | 头像区 | 大头像+昵称+排名徽章 |
| 个人资料页面 | 统计网格 | 2x4网格展示各项数据 |
| 个人资料页面 | 成就墙 | 徽章网格，已解锁高亮 |

### 4.3 响应式
- 桌面优先，Canvas自适应窗口大小
- 支持键盘（WASD/方向键）+ 鼠标双操控
- 排行榜和资料页面响应式适配

### 4.4 3D场景指引
不适用，本项目为2D Canvas游戏。
